A Drupal module for managing pharmaceutical establishments.

## Installation

1. Place this module in `/modules/custom/`
2. Enable the module via Drupal admin or using Drush:
   ```
   drush en etablissements_pharmaceutiques
   ```

## Features

- Manage pharmaceutical establishments
- Custom content type for pharmaceutical establishments

## Usage

After enabling the module, you can create and manage pharmaceutical establishment content through the Drupal admin interface.
